# 🧠 Real-Time Whiteboard App

A full-stack whiteboard application that enables users to draw and collaborate in real time using web technologies.

---

## 🚀 Features

- ✏️ Canvas-based drawing system  
- 🔄 Real-time updates using WebSockets  
- 🧩 Modular UI components (Board, Toolbar, Toolbox)  
- 🧹 Clear canvas functionality  
- ⚡ Smooth drawing experience  

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- HTML5 Canvas
- CSS 

**Backend**
- Node.js
- Express.js
- Socket.io

---

## 📁 Project Structure
